# Cyrylica od zera

Statyczna aplikacja PWA do nauki cyrylicy dla osób polskojęzycznych.

## Uruchomienie

Otwórz `index.html` w przeglądarce.

Do działania offline i instalacji PWA hostuj folder przez HTTP/HTTPS, np. na GitHub Pages, Netlify, Cloudflare Pages albo dowolnym statycznym hostingu. Aplikacja nie wymaga serwera, bazy danych, logowania ani bibliotek zewnętrznych.

## Funkcje

- teoria i plan nauki od zera
- kolejność alfabetu rosyjskiego i podstawy układu klawiatury ЙЦУКЕН
- informacja o różnicach między rosyjskim, ukraińskim i białoruskim alfabetem cyrylickim
- aktywna tabela liter
- wielkie i małe litery
- wybór poziomu oraz zestawów znaków do ćwiczeń
- tryby prezentacji znaków: wielkie, małe, mieszane albo para
- przykłady użycia znaków w słowach
- klikalne przykłady z odsłuchem konkretnego słowa
- lokalna grafika rosyjskiej klawiatury ЙЦУКЕН
- fiszki adaptacyjne
- quiz wyboru
- quiz słowny w obu kierunkach: zapis łaciński → cyrylica oraz cyrylica → zapis łaciński
- progresja słów według etapów trudności 0-3 albo tryb mieszany
- tryb pisania
- łączenie par
- lokalny postęp w `localStorage`
- dźwięki przez Web Audio API
- opcjonalny odsłuch rosyjskich przykładów przez `speechSynthesis`
- responsywny układ dla desktopu i Safari na iOS
